# 🐧 Coding Journey for Kids: Linux, Node.js & Golang

## About This Plan

- **Student:** 9-year-old beginner, no prior coding experience
- **Setup:** Linux Mint laptop
- **Tools:** Google Drive for sharing lesson plans
- **Duration:** ~22 weeks (~5–6 months)
- **Session Length:** 30–45 minutes recommended

---

## Phase 1: Linux Basics (Weeks 1–6)

**Goal:** Get comfortable navigating and using Linux Mint.

### Week 1–2: Getting to Know Linux
- What is an operating system?
- Using the desktop (files, folders, apps)
- Opening and using the Terminal
- Basic commands: `ls`, `cd`, `pwd`, `mkdir`, `rm`, `cp`, `mv`

### Week 3–4: Files & Permissions
- Creating and editing text files with `nano`
- Understanding file permissions (`chmod`)
- Users and home directories
- Installing software with `apt`

### Week 5–6: Bash Shell Scripting
- What is a script?
- Writing your first `.sh` file
- Variables, `echo`, and user input
- `if` statements and loops (`for`, `while`)
- Making scripts executable

### 🛠 Phase 1 Projects
| Project | Description |
|---|---|
| My Info Script | Asks your name and prints a fun greeting |
| Folder Organizer | Sorts files into folders by type |
| Guessing Game | Number guessing game in bash |

---

## Phase 2: Node.js (Weeks 7–14)

**Goal:** Learn JavaScript programming through Node.js.

### Week 7–8: JavaScript Basics
- What is JavaScript and Node.js?
- Running your first `.js` file in the terminal
- Variables (`let`, `const`), data types, and `console.log`
- Math and strings

### Week 9–10: Logic & Control Flow
- `if/else` statements
- Loops (`for`, `while`)
- Functions — writing and calling them

### Week 11–12: Working with Data
- Arrays and objects
- Looping through arrays
- Reading user input with `readline`

### Week 13–14: Building Small Apps
- Introduction to `npm` (Node Package Manager)
- Using simple packages
- Writing programs that do real things

### 🛠 Phase 2 Projects
| Project | Description |
|---|---|
| Mad Libs Generator | Fill-in-the-blank story using user input |
| Simple Calculator | Add, subtract, multiply, divide from the terminal |
| To-Do List App | Add/remove tasks, save to a file |
| Trivia Quiz Game | Ask questions, keep score, show results |

---

## Phase 3: Golang (Weeks 15–22)

**Goal:** Learn Go as a first compiled, statically typed language.

### Week 15–16: Hello, Go!
- What makes Go different from JavaScript?
- Installing Go and setting up the workspace
- Your first `main.go` program
- `fmt.Println`, variables, and types

### Week 17–18: Go Fundamentals
- Integers, strings, booleans
- `if/else`, `switch` statements
- `for` loops (Go's only loop!)
- Functions with return values

### Week 19–20: Data Structures
- Arrays and slices
- Maps (like objects in JS)
- Structs — creating your own data types

### Week 21–22: Building Real Programs
- Reading user input with `bufio`
- Writing and reading files
- Basic error handling
- Putting it all together

### 🛠 Phase 3 Projects
| Project | Description |
|---|---|
| Temperature Converter | Convert Fahrenheit to Celsius and back |
| Word Counter | Count words in a sentence the user types |
| Animal Encyclopedia | Store and look up animal facts using structs and maps |
| Text Adventure Game | Simple choose-your-own-adventure story |

---

## Tips for Teaching a 9-Year-Old
- Keep sessions to **30–45 minutes** max
- Always connect concepts to something **fun or real** (games, stories, animals)
- Celebrate every working program, no matter how small
- Let him break things and figure out why — that's real learning
- Share one lesson plan at a time via Google Drive so it doesn't feel overwhelming
